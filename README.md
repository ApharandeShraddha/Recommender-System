An information filtering system that seek to predict the "rating" or "preference" that a user would give to an item.
Used collaborative filtering method to collect and analyze a large amount.
Implemented a new algorithm using combination of k-nearest neighbor (k-NN) approach and the Pearson Correlation method.
Used the the intelligent model which uses given training set and test set in order to reduce effects of overfitting / underfitting.
Such recommendation systems are used by sellers like amazon to suggest products to user
